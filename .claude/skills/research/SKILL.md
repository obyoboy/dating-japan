# Skill: research — 記事テーマ・キーワードリサーチ

## 概要
`en/` フォルダの既存記事と重複しない新しい記事テーマを1つ選定し、3言語分のタイトル・slug・キーワードをJSON形式で出力する。

---

## 手順

### Step 1: 既存記事の確認
`en/` フォルダのHTMLファイル一覧を取得し、カバー済みトピックを把握する。

```bash
ls en/*.html
```

### Step 2: テーマ選定
以下の候補から、**既存記事と重複しない**テーマを1つ選ぶ。
ロングテール・低競合キーワードを優先する。

**テーマ候補プール:**
- 日本人女性のデートへの誘い方
- 日本のデート文化を理解する
- 初デートにおすすめの場所・プラン
- 外国人のための日本のデートエチケット
- 日本のバレンタインデー・ホワイトデーの文化
- 告白（こくはく）の仕組みと成功するタイミング
- 日本人女性が外国人に求めるもの
- 日本語でのデート表現・フレーズ集
- マッチングアプリで日本人女性に出会う方法
- 日本人女性の家族に紹介される準備
- 長距離恋愛・国際カップルの注意点
- 日本のデートでのNG行動・あるある失敗集

候補プールが尽きた場合は、ターゲット読者（外国人男性）に役立つ新テーマを自ら提案する。

### Step 3: JSON出力
以下のフォーマットで出力する（コードブロックなし・JSONのみ）:

```json
{
  "topic": "日本語でテーマ説明",
  "primary_keyword": "英語メインキーワード（ロングテール）",
  "secondary_keywords": ["関連キーワード1", "関連キーワード2", "関連キーワード3"],
  "slug": "ハイフン区切り英小文字-slug",
  "badge_category": "バッジカテゴリ名",
  "title_en": "英語タイトル",
  "title_tl": "タガログ語タイトル",
  "title_vi": "ベトナム語タイトル"
}
```

---

## バッジカテゴリ一覧
`Culture` / `First Dates` / `Communication` / `Relationships` / `Etiquette` / `Where to Meet` / `Language` / `Tips & Warnings`

---

## 注意事項
- slugは既存ファイル名と被らないこと
- タイトルは3言語とも**翻訳ではなく自然な表現**で書く
- primary_keywordは読者が実際に検索しそうな英語フレーズにする
- コンテンツは誠実で尊重ある内容にする（ステレオタイプや不適切な表現を避ける）
